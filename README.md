# Job Board

A MERN stack job board application where users can:
- View job listings
- Add new jobs
- Edit jobs
- Delete jobs

## Installation
1. Clone the repository
2. Run `npm install` in both `Frontend` and `Backend` folders
3. Run `npm run dev` for frontend, `npm start` for backend
